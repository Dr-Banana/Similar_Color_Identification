<p class=IEEETitle><b style='mso-bidi-font-weight:normal'><span lang=EN-AU
style='font-size:18.0pt;font-family:"Cambria",serif;mso-fareast-font-family:
Cambria;mso-bidi-font-family:Cambria;letter-spacing:-.3pt'>Similar <span
class=SpellE>Color</span> Identification Using Number of Cluster Algorithm<o:p></o:p></span></b></p>

<p class=IEEETitle><b style='mso-bidi-font-weight:normal'><span lang=EN-AU
style='font-size:12.0pt;font-family:"Cambria",serif;mso-fareast-font-family:
Cambria;mso-bidi-font-family:Cambria;letter-spacing:-.3pt'><o:p>&nbsp;</o:p></span></b></p>

<p class=IEEETitle><b style='mso-bidi-font-weight:normal'><span lang=EN-AU
style='font-size:12.0pt;font-family:"Cambria",serif;mso-fareast-font-family:
Cambria;mso-bidi-font-family:Cambria;letter-spacing:-.3pt'>Xuanzhi Zhang<o:p></o:p></span></b></p>

<p class=Affiliation><span style='font-size:9.0pt'><o:p>&nbsp;</o:p></span></p>

<p class=Affiliation style='text-indent:.5in'><span style='font-size:9.0pt'>Department
of Physics,</span> <span style='font-size:9.0pt'>The Ohio State University<o:p></o:p></span></p>

<p class=Affiliation><span style='font-size:9.0pt'>zhang.10419@osu.edu<o:p></o:p></span></p>

<p class=Affiliation><span style='font-size:11.0pt'><o:p>&nbsp;</o:p></span></p>

<p class=Affiliation><span style='font-size:11.0pt'>Project Advisor: Dr. Marc <span
class=SpellE>Bockrath</span>, Department of Physics<o:p></o:p></span></p>

<p class=MsoNormal align=center style='text-align:center'><span
style='font-size:11.0pt'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><b style='mso-bidi-font-weight:normal'><span
style='font-family:"Times New Roman",serif;mso-fareast-font-family:Cambria;
mso-font-width:102%'>ABSTRACT<o:p></o:p></span></b></p>

<p class=MsoNormal><span style='font-size:11.0pt;font-family:"Times New Roman",serif'>With
the rapid development of 2D materials and their applications in the industrial
area in recent years, the automatic search for mechanically exfoliated van der
Waals via optical imaging has become a crucial part of improving efficiency.
However, currently, the Deep Learning method widely used by computer vision and
the traditional Machine Learning method cannot be adapted to materials whose
color is too <span class=GramE>similar to</span> the background. Also, too many
image data sets are required for training a model. In this paper, we introduce
the Number of Cluster (<span class=SpellE>NoC</span>) algorithm to quickly
calculate the target number k required by K-Means clustering algorithm</span><span
style='font-size:11.0pt;font-family:"Times New Roman",serif;mso-fareast-font-family:
宋体;mso-fareast-language:ZH-CN'>. </span><span style='font-size:11.0pt;
font-family:"Times New Roman",serif'>This research can not only be used for the
identification of two-dimensional materials on SiO2/Si substrate, but also
widely used for color extraction of various continuous/discontinuous areas
under a single-color background. Eight pictures with different number of
discrete color areas were used to test the time and accuracy of k value
calculation by the Gap statistic method<sup>[13]</sup>, Elbow method<sup>[14]</sup>,
and <span class=SpellE>NoC</span> method respectively. <span
style='mso-spacerun:yes'>&nbsp;</span>The result shows that the <span
class=SpellE>NoC</span> method runtime only takes 0.23444 seconds to test with 52*52
discrete area images.<o:p></o:p></span></p>

<p class=MsoNormal><b style='mso-bidi-font-weight:normal'><span
style='font-family:"Times New Roman",serif;mso-fareast-font-family:Cambria;
mso-font-width:102%'>INTRODUCTION<o:p></o:p></span></b></p>

<p class=MsoNormal align=left style='margin-bottom:0in;text-align:left;
line-height:normal;mso-layout-grid-align:none;text-autospace:none'><span
style='font-size:11.0pt;font-family:"Times New Roman",serif;mso-fareast-font-family:
等线;mso-fareast-theme-font:minor-fareast;mso-fareast-language:ZH-CN'>The
remarkable properties of graphene, including its electrical<sup>[1]</sup>,
ferromagnetic<sup>[2,3]</sup>, semiconductor<sup>[4]</sup>, superconductivity<sup>[5]</sup>,
and optical properties<sup>[6,7]</sup>, have made it an attractive material for
scientific and industrial electronic components. Currently, mechanical
exfoliation using tape is the primary method for obtaining high-quality
graphene samples. However, this method produces random distributions of edge
shapes and layers. Fortunately, different layers of graphene exhibit unique
color and wavelength responses at a thickness of 270 nm on a SiO2/Si substrate<sup>[8]</sup>,
which can be observed using an optical microscope. Therefore, shape information
cannot be used as the variable of layer classification, and only color
information can be used to judge layer number and boundary of each layer. In
this work, we propose a technique that represents different layers of graphene
in microscopic images using distinct colors and introduces an algorithm to
rapidly determine k values for K-Means from the images.<o:p></o:p></span></p>

<p class=MsoNormal align=left style='margin-bottom:0in;text-align:left;
line-height:normal;mso-layout-grid-align:none;text-autospace:none'><span
style='font-size:11.0pt;font-family:"Times New Roman",serif;mso-fareast-font-family:
等线;mso-fareast-theme-font:minor-fareast;mso-fareast-language:ZH-CN'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal align=left style='margin-bottom:0in;text-align:left;
line-height:normal;mso-layout-grid-align:none;text-autospace:none'><span
style='font-size:11.0pt;font-family:"Times New Roman",serif;mso-fareast-font-family:
等线;mso-fareast-theme-font:minor-fareast;mso-fareast-language:ZH-CN'>Because of
the increasing variety of two-dimensional (2D) materials, each laboratory
microscope can photograph the same material in very different colors.
Therefore, an algorithm that can calculate k value quickly and accurately is
urgently needed, and the calculation time of k value needs to be independent of
the size of the image and the number of color blocks in the image.<o:p></o:p></span></p>

<p class=MsoNormal align=left style='margin-bottom:0in;text-align:left;
line-height:normal;mso-layout-grid-align:none;text-autospace:none'><span
style='font-size:11.0pt;font-family:"Times New Roman",serif;mso-fareast-font-family:
等线;mso-fareast-theme-font:minor-fareast;mso-fareast-language:ZH-CN'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal align=left style='margin-bottom:8.0pt;text-align:left;
line-height:107%'><span style='font-size:11.0pt;line-height:107%;font-family:
"Times New Roman",serif;mso-fareast-font-family:等线;mso-fareast-theme-font:minor-fareast;
mso-fareast-language:ZH-CN'>Features extraction is a crucial step in
identifying and interpreting meaningful information from image data<sup>[9]</sup>.
To distinguish between areas with similar colors. The <span class=SpellE>NoC</span>
algorithm accomplishes this by transforming the linear relationship between
layers and the RGB color space into a nonlinear relationship, which can
maximize the differentiation between the colors of different layers. During our
experiments, we found that the L value of the CIELAB color space<sup>[20] </sup>(LAB)
represents the lightness of the color, which can be used to determine the
number of graphene layers represented by a pixel. Additionally, the a and b
values can be utilized to distinguish between different materials based on
their color properties. <o:p></o:p></span></p>

<p class=MsoNormal align=left style='margin-bottom:8.0pt;text-align:left;
line-height:107%'><span style='font-size:11.0pt;line-height:107%;font-family:
"Times New Roman",serif;mso-fareast-font-family:等线;mso-fareast-theme-font:minor-fareast;
mso-fareast-language:ZH-CN'>As we did not use any training data, we employed
the unsupervised machine learning K-Means<sup>[19]</sup> algorithm for image
region segmentation. K-means is a popular unsupervised machine learning
algorithm used for clustering data into distinct groups based on similarities
in their features. In K-means, the user specifies the number of clusters, k,
and the algorithm assigns each data point to one of these k clusters based on
the similarity of its features to the centroids of the clusters.</span> </p>

<p class=MsoNormal align=left style='margin-bottom:8.0pt;text-align:left;
line-height:107%'><span style='font-size:11.0pt;line-height:107%;font-family:
"Times New Roman",serif;mso-fareast-font-family:等线;mso-fareast-theme-font:minor-fareast;
mso-fareast-language:ZH-CN'>The most popular Gap statistic method and Elbow
method as algorithms to automatically calculate k values require manual input
of a maximum k value. However, the larger the maximum value is, the longer the runtime
will be. In contrast, the <span class=SpellE>NoC</span> algorithm can calculate
the k value without any preset maximum k value or condition. By utilizing a
nonlinear equation to analyze LAB information, we can determine the number of
color types in the image, which corresponds to the value of k required for
K-Means clustering. This approach eliminates the need for manual intervention
and makes the process more efficient.<span style='color:red'><o:p></o:p></span></span></p>
